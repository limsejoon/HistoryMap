# 지도 웹페이지 디자인 스펙

## 개요

Leaflet.js + OpenStreetMap 타일을 사용한 단일 HTML 파일 지도 앱. 회원가입·API 키·결제 불필요.

---

## 핵심 결정사항

| 항목 | 결정 |
|------|------|
| 지도 라이브러리 | Leaflet.js (CDN) |
| 타일 소스 | OpenStreetMap (무료, API 키 불필요) |
| 초기 뷰 | 세계 전체 (center: [20, 0], zoom: 2) |
| 마커 추가 방식 | 지도 클릭 → 즉시 추가 |
| 팝업 내용 | 위도/경도 자동 표시 (소수점 4자리) |
| 마커 삭제 | 팝업 내 "삭제" 버튼 |
| 데이터 저장 | 없음 (새로고침 시 초기화) |
| 구현 방식 | 단일 HTML/CSS/JS 파일 |

---

## 화면 흐름

```
[초기 세계지도] → 지도 클릭 → [마커 추가]
                                    ↓
                              마커 클릭
                                    ↓
                         [팝업: 좌표 + 삭제 버튼]
                                    ↓
                              삭제 버튼 클릭
                                    ↓
                            [마커 제거]
```

---

## 화면 구성

### 지도 영역
- 화면 전체(`100vw × 100vh`)를 지도로 채움
- Leaflet 기본 줌 컨트롤(+/−) 우상단 표시
- 하단 안내 문구: "지도를 클릭하면 마커가 찍힙니다"
- 하단 OpenStreetMap 저작권 표시 (Leaflet 자동 처리)

### 마커
- Leaflet 기본 파란 마커 아이콘 사용
- 클릭 이벤트로 추가, 별도 상태 저장 없이 Leaflet 레이어로만 관리

### 팝업
- 마커 클릭 시 Leaflet 기본 팝업 표시
- 내용: 위도/경도 (소수점 4자리, 예: `37.5665° N, 126.9780° E`)
- 삭제 버튼: 빨간색, 클릭 시 해당 마커 제거 후 팝업 닫힘

## 기술 구조

```html
<!DOCTYPE html>
<html>
<head>
  <!-- Leaflet CSS (CDN) -->
  <!-- 인라인 CSS: body/html 100%, #map 100vh -->
</head>
<body>
  <div id="map"></div>
  <!-- Leaflet JS (CDN) -->
  <script>
    // 1. 지도 초기화 (세계 뷰)
    // 2. OSM 타일 레이어 추가
    // 3. 지도 클릭 핸들러 → 마커 추가
    // 4. 마커 팝업 HTML (좌표 + 삭제 버튼)
    // 5. 삭제 버튼 이벤트 → 마커 제거
  </script>
</body>
</html>
```

---

## 상태 관리

별도 JS 상태 없음. 마커는 Leaflet 레이어로만 관리:
- 추가: `L.marker([lat, lng]).addTo(map)`
- 삭제: `map.removeLayer(marker)`
